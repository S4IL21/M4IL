Welcome to the M4IL!

M4IL is a simple web application that allows you to generate temporary email addresses and view the emails received by these addresses.

This project uses 1secmail's API for creating e-mails.

⚠️ This project is purely for educational purposes. We do not allow illegal activities to be performed with this project and we are not responsible for any incidents that may occur. Use it legally ⚠️
